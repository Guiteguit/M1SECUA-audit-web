# J2-C03 — Analyse DOM/API avec Juice Shop

## Objectif
Comprendre comment une application moderne charge ses données côté navigateur.

## Cible autorisée
`http://juice.target.local:3000`

## Consignes
1. Ouvrir DevTools avec F12.
2. Onglet `Network`, filtrer `Fetch/XHR`.
3. Recharger la page.
4. Faire une recherche produit.
5. Identifier au moins un endpoint API.
6. Observer `Application/Storage`.
7. Chercher dans `Sources` : `token`, `authorization`, `localStorage`, `innerHTML`, `redirect`.

## Preuve attendue
Endpoint API, stockage navigateur, observation DOM, hypothèse de risque, recommandation.
