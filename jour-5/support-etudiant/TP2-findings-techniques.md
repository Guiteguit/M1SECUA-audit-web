# TP2 — Findings techniques détaillés

## Objectif

Rédiger des findings professionnels et reproductibles.

## Consigne

Rédiger au minimum 5 findings détaillés :

- 1 SQL Injection ;
- 1 XSS ;
- 1 IDOR ou contrôle d’accès ;
- 1 JWT ou authentification ;
- 1 finding SAST / code review.

## Structure attendue

Chaque finding doit contenir :

- titre clair ;
- sévérité ;
- score CVSS ;
- CWE ;
- endpoint ou fichier ;
- paramètre ;
- description ;
- étapes de reproduction ;
- preuve ;
- impact ;
- cause racine ;
- remédiation ;
- vérification attendue.
