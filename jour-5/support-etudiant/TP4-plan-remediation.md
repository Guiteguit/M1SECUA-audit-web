# TP4 — Plan de remédiation

## Objectif

Transformer les findings en plan d’action réaliste.

## Consigne

Construire une roadmap :

- actions immédiates : 0 à 7 jours ;
- court terme : 2 à 4 semaines ;
- moyen terme : 1 à 3 mois ;
- long terme : industrialisation.

## Exemples

| Priorité | Action |
|---|---|
| P0 | Corriger SQL Injection avec requêtes préparées |
| P0 | Refuser JWT alg=none |
| P1 | Ajouter tests d’autorisation IDOR |
| P1 | Ajouter protection CSRF |
| P2 | Intégrer SAST dans CI/CD |
| P2 | Former les développeurs au secure coding |
