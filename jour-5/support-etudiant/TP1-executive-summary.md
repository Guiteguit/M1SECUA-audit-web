# TP1 — Executive Summary

## Objectif

Rédiger un résumé exécutif compréhensible par un décideur non technique.

## Consigne

À partir des findings des Jours 1 à 4, rédiger une page maximum qui explique :

- le niveau de risque global ;
- les principaux risques ;
- les impacts métier ;
- les actions immédiates ;
- les besoins de remédiation.

## À éviter

- trop de détails techniques ;
- payloads ;
- jargon non expliqué ;
- longues sorties Burp ou logs.

## À produire

Compléter :

```text
livrables/01-executive-summary.md
```

Puis intégrer la version consolidée dans :

```text
livrables/rapport-final.md
```

## Questions d’aide

1. Quels risques peuvent impacter la confidentialité ?
2. Quels risques peuvent permettre une prise de contrôle ?
3. Quels risques doivent être corrigés en premier ?
4. Comment expliquer le risque en termes simples ?
