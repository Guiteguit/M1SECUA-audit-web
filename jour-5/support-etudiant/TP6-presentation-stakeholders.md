# TP6 — Présentation aux stakeholders

## Objectif

Préparer une restitution orale de 10 à 15 minutes.

## Consigne

Créer une présentation courte à partir de :

```text
templates/slides-restitution-template.md
```

Compléter :

```text
livrables/slides-restitution.md
```

## Structure attendue

1. Contexte et périmètre
2. Risk dashboard
3. Top 3 findings
4. Roadmap de remédiation
5. Secure coding tips
6. Questions

## Conseil

Pour le management :

- parler impact ;
- parler priorité ;
- éviter trop de payloads ;
- expliquer le coût de non-correction.

Pour les développeurs :

- montrer la cause racine ;
- proposer un correctif ;
- donner un test de vérification.
