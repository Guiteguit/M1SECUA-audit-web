# J4-C03 — WebGoat guided review

## Objectif
Consolider deux familles de vulnérabilités avec WebGoat.

## Consignes
Choisir deux familles parmi :
- Access Control ;
- Authentication ;
- JWT ;
- XXE ;
- Injection ;
- XSS.

Pour chaque famille, documenter : objectif, preuve de réussite, vulnérabilité, impact, remédiation, contrôle automatisable ou manuel.
