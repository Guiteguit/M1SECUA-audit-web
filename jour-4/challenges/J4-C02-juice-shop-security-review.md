# J4-C02 — Mini revue sécurité Juice Shop

## Objectif
Produire une mini cartographie sécurité d’une application moderne.

## Consignes
1. Ouvrir Juice Shop.
2. Capturer au moins 3 appels API dans DevTools/Burp.
3. Identifier les méthodes HTTP.
4. Observer le stockage navigateur.
5. Formuler 3 hypothèses de risque.
6. Proposer 3 recommandations.

## Preuve attendue
Endpoints API, stockage, hypothèses, recommandations.
